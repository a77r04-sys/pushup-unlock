💪 Push-up Unlock

Instagram & X stay locked until the camera sees real push-ups.

Prop the phone to your side, ~2 m away, whole body in frame.

📱 One-time iPhone setup (Shortcuts)

This page only counts push-ups. Two Shortcuts + two Automations make it gate Instagram/X. Everything below is done in the built-in Shortcuts app — no other downloads, no accounts.

Shortcut 1 — "Pushups Done" (grants 30 min)

  1. Shortcuts app → + → name it exactly Pushups Done (must match the name in the box above).
  2. Add action Date → Current Date.
  3. Add action Adjust Date → "Add 30 minutes to Date". (This is your unlock length — change it here.)
  4. Add action Format Date → Date Format: ISO 8601, Time: ISO 8601 (include time).
  5. Add action Save File → turn Ask Where To Save OFF → Service: iCloud Drive → path /Shortcuts/pushup-unlock.txt → turn Overwrite If File Exists ON.
  6. Add action Open App → Instagram.

Shortcut 2 — "Pushup Gate" (runs when you open the app)

  1. New shortcut, name it Pushup Gate.
  2. Add action Get File → iCloud Drive → path /Shortcuts/pushup-unlock.txt → turn Show Document Picker OFF, and turn ON "Error If Not Found" → then tap the action's arrow and wrap it: if it errors the first run, that's fine — see step 5.
  3. Add action Get Dates from Input (input: File).
  4. Add action If → Dates → Condition: is in the past … iOS wording: choose If [Date] is before [Current Date].
  5. Inside If (expired → locked): add Open URLs → THIS_PAGE_URL (long-press → copy this page's address from Safari). Inside Otherwise: nothing — the app just opens. If the file doesn't exist yet, Shortcuts treats it as locked, which is correct.

Automations (the actual gate)

  1. Shortcuts app → Automation tab → + → App → Choose Instagram → "Is Opened" → Run Immediately (not "Ask Before Running").
  2. Next screen: pick shortcut Pushup Gate.
  3. Repeat for the X app.

How it flows

Open Instagram → automation runs the Gate → unlock expired → Safari opens this page over the app → do your push-ups → the page runs Pushups Done → 30-min pass is written → Instagram reopens. When the pass expires, the next app-open kicks you back here. To pause the whole system, toggle the two automations off in the Shortcuts app.

Honest limitations

Like the Mac version, this is friction, not a prison: you can swipe back to the app quickly or disable the automation. iOS won't let a homemade phone-only tool hard-block apps — that needs Apple's Screen Time entitlement. For harder enforcement, add a Screen Time app limit with someone else holding the passcode, on top of this.
