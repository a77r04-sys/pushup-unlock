📱 One-time iPhone setup (Shortcuts)

This page only counts push-ups. Two Shortcuts + two Automations make it gate Instagram/X. Everything below is done in the built-in Shortcuts app — no other downloads, no accounts.

Shortcut 1 — "Pushups Done" (saves your earned pass)

This page tells the Shortcut how many minutes you earned (push-ups × minutes-per-push-up), so the Shortcut must use its input instead of a fixed number.

  1. Shortcuts app → + → name it exactly Pushups Done (must match the name in the box above).
  2. Add action Adjust Date. Tap the date → choose Current Date. Tap the number → Select Variable → Shortcut Input. Set the unit to minutes, so it reads "Add Shortcut Input minutes to Current Date".
  3. Add action Format Date → Date Format: ISO 8601, Time: ISO 8601 (include time).
  4. Add action Save File → turn Ask Where To Save OFF → Service: iCloud Drive → path /Shortcuts/pushup-unlock.txt → turn Overwrite If File Exists ON.

Do not add an Open App action — this page reopens your app itself. Already built the old version? Just edit it: replace the fixed "30" in Adjust Date with the Shortcut Input variable, and delete the Open App step.

Shortcut 2 — "Pushup Gate" (runs when you open the app)

  1. New shortcut, name it Pushup Gate.
  2. Add action Get File → iCloud Drive → path /Shortcuts/pushup-unlock.txt → turn Show Document Picker OFF.
  3. Add action Get Dates from Input (input: File).
  4. Add action If → Dates → Condition: is in the past … iOS wording: choose If [Date] is before [Current Date].
  5. Inside If (expired → locked): add Open URLs → THIS_PAGE_URL (long-press → copy this page's address from Safari). Inside Otherwise: nothing — the app just opens. If the file doesn't exist yet, Shortcuts treats it as locked, which is correct.

Automations (the actual gate)

  1. Shortcuts app → Automation tab → + → App → Choose Instagram → "Is Opened" → Run Immediately (not "Ask Before Running").
  2. Next screen: pick shortcut Pushup Gate.
  3. Repeat for the X app.

How it flows

Open Instagram → automation runs the Gate → pass expired → Safari opens this page over the app → do your push-ups → the page runs Pushups Done with your earned minutes → the pass is written → your app reopens. When the pass expires, the next app-open kicks you back here. The Block apps toggle above uses the same mechanism: off writes a ~10-year pass, on writes an already-expired one.

Honest limitations

Like the Mac version, this is friction, not a prison: you can swipe back to the app quickly or disable the automation. iOS won't let a homemade phone-only tool hard-block apps — that needs Apple's Screen Time entitlement. For harder enforcement, add a Screen Time app limit with someone else holding the passcode, on top of this.
